Encodes a chat packet into a string given username, id, timestamp, and message. Intended packet format: 3,username,id,timestamp,message\r [0,1 ,2 ,3 ,4 ]

Examples

Testing the encodeChatPacket() method.

string testUsername = "User";

long testTimeStamp = 125;
string testMessage = "This is a test message.";

string testChatPacket = encodeChatPacket(testUsername, testId, testTimeStamp, testMessage);
